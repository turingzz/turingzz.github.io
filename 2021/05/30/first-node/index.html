Node安装和配置国内镜像

Author: turingzz Date: May 30, 2021  16:12:53 Category: study

安装node之后，不能只是在简单的命令行开始学习。

node在控制台的语法和普通的js没什么差别，写过大量的js直接从服务器开始摸索。

首先，习惯了Java里面的配置，刚刚运行一个node服务端的小例子，感觉到了和php一样的简便，npm安装一个supervisor，自动重新部署，更有PHP的感觉了。

  • 安装node

一步步安装下来，会自动配置系统变量

  • npm设置淘宝镜像

输入npm config set registry https:*//registry.npm.taobao.org*

验证是否设置成功npm config get registry

  • 安装supervisor热部署

npm install -g supervisor

安装之后，之前使用node xxx.js，现在使用supervisor xxx.js

*如果spuervisor启动js文件时，提示无法加载文件。那么用管理员身份运行Powershell，输入命令set-ExecutionPolicy RemoteSigned，输入Y回车。
